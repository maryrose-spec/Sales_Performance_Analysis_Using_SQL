# Sales Performance Analysis using SQL

## Project Overview

This project analyzes sales data using SQL.

## Tools Used

- SQL (MySQL)
- Excel

## Business Questions

1. Which category generated the most revenue?
2. Which Sales Channel Generated Most Revenue?
3. Which region performed best?
4. Which employee handled the most orders?
5. What are the monthly sales trends?
6.What is the Total Revenue?
7. Highest Selling Category by Quantity
8. Which Product Category Generates the Highest Revenue Despite Offering Discounts?

## SQL Skills Demonstrated

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- Aggregate Functions
