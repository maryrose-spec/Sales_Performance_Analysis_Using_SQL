# Sales Performance Analysis using SQL

## Project Overview

This project analyzes sales data using SQL.

## Tools Used

- SQL (MySQL)
- Excel

## Business Questions

1. What is the total revenue?
2. Which category generated the most revenue?
3. Which region performed best?
4. Which employee handled the most orders?
5. What are the monthly sales trends?

## SQL Skills Demonstrated

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- Aggregate Functions
